🌍 City Explorer

A smart Flask-based web app that gives real-time information about any city including weather, news, tourist attractions, local time and location details.

🚀 Features :

 Real-time weather data, 
 Latest city-related news, 
 Tourist attractions nearby, 
 local time, 
 AI-generated city description, 
 Location details (state, country), 
 Wikipedia city summary + image, 
Clean and interactive UI.

🛠 Tech Stack : 

Python (Flask), 
HTML, CSS, JavaScript, 
OpenWeather API, 
Geoapify API, 
NewsAPI, 
Wikipedia REST API.
